## Getting Started

* Clone the repository
* Run bundle install
* Create and migrate the PotgreSQL database with rake db:create and rake db:migrate
* Start the server using rails server -b 0.0.0.0 -p 3000
* Run the app on localhost:3000

### Prerequisites

* Rails v. 5.0.0
* Ruby v. 2.6.3
* JavaScript
* PostgreSQL
* HTML
* CSS/SCSS
